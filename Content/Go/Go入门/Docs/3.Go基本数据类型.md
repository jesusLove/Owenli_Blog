# 数据类型

## 基本数据类型

1. 整数类型：`int`、`int32`、`int64`等
2. 浮点类型: `float32`、`float64`。没有`float`类型 。
3. 布尔类型：`true`、`false`。
4. 字符串：`string`，两个包:`strings`字符串操作；`strconv`类型转换。

## 数组

```go
var arr = [5]int{0, 1, 2, 3, 4}
fmt.Println(arr))
```
